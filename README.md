# KaryaSarthi - Empowering Job Seekers and Employers

> **🚧 This project is still a work in progress! 🚧**  
> We're building a revolutionary job portal, and we’re excited to bring you on the journey! Stay tuned for updates as we develop and enhance the platform.


KaryaSarthi is a full-stack job portal designed to connect job seekers with employers, providing a seamless platform to create profiles, search and apply for jobs, and manage job postings. This platform leverages modern web technologies to deliver an intuitive and responsive user experience.

## 🚀 Features

- **Job Seeker:**
  - Create and manage profile.
  - Search and filter jobs by location, title, and industry.
  - Apply to job listings with a single click.

- **Employer:**
  - Post and manage job listings.
  - Review applications from job seekers.
  - Edit or remove job listings as needed.

- **Responsive Design:** 
  - Fully responsive for mobile, tablet, and desktop views.
  
- **User Authentication:** 
  - Secure login system with Clerk for role-based access control.

## 🧑‍💻 Tech Stack

- **Frontend:**
  - React.js
  - Tailwind CSS
  - Shadcn UI
- **Backend:**
  - MongoDB
  - Express.js
- **Authentication:**
  - Clerk
